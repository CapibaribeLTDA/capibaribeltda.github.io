# Política de Privacidade e Proteção de Dados

---

## 1. Quais Dados Coletamos?

Coletamos os seguintes tipos de dados pessoais:

- **Dados fornecidos voluntariamente**:  
  Nome, e-mail, telefone, CPF, endereço, data de nascimento, etc., quando você se cadastra em nossos serviços ou interage conosco.

- **Dados de navegação**:  
  IP, cookies, dados de localização, tipo de dispositivo, páginas acessadas e tempo de permanência (via tecnologias como Google Analytics).

- **Dados de transações**:  
  Histórico de compras, pagamentos e preferências de consumo (se aplicável).

---

## 2. Finalidades do Tratamento de Dados

Seus dados serão utilizados para:

- Prestar serviços contratados ou enviar comunicações necessárias.
- Personalizar sua experiência e melhorar nossos produtos.
- Cumprir obrigações legais ou regulatórias.
- Enviar newsletters ou promoções (mediante consentimento).

---

## 3. Bases Legais (LGPD)

O tratamento de dados é baseado em:

- **Consentimento** (Art. 7º, I da LGPD): Para finalidades específicas, com direito a revogação a qualquer momento.
- **Contrato** (Art. 7º, V): Quando necessário para execução de serviços.
- **Interesse legítimo** (Art. 7º, IX): Para melhorias operacionais ou segurança.

---

## 4. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

- **Parceiros de negócios**: Apenas para fins contratuais (ex.: processamento de pagamentos).
- **Autoridades legais**: Em cumprimento a ordens judiciais ou regulatórias.
- **Terceiros**: Anonimizados para análises estatísticas.

---

## 5. Direitos do Titular (LGPD)

Você tem o direito de:

- Acessar, corrigir ou solicitar a exclusão dos seus dados.
- Revogar consentimentos.
- Portar seus dados para outro fornecedor.
- Opôr-se a tratamentos não autorizados.

**Como exercer seus direitos**:  
Envie um e-mail para [e-mail do Encarregado de Dados (DPO)] com o assunto "Direitos LGPD".

## 6. Segurança dos Dados

Adotamos medidas técnicas e administrativas para proteger seus dados, incluindo:

- Criptografia de informações sensíveis.
- Acesso restrito a colaboradores autorizados.

Seus dados serão armazenados apenas pelo tempo necessário para as finalidades descritas ou exigido por lei. Após isso, serão anonimizados ou excluídos.
